# VidArchive

VidArchive is a desktop utility application for downloading video and audio through an interactive user interface.

Media extraction is handled via integration with yt-dlp and ffmpeg.

[![Watch the demo](https://res.cloudinary.com/dqzusd5rw/image/upload/v1767723060/Screenshot_2026-01-06_234048_brlxby.png)](https://res.cloudinary.com/dqzusd5rw/video/upload/v1767723011/VidArchive_0.21.0_demo_final_col1yb.mp4)

## Tech Stack

- Electron.js
- React.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- better-sqlite3
- Drizzle ORM
- Zustand

## Features

- Download video and audio media
- View media metadata
- Interactively select formats and options
- URL and download history
- Application settings

## Supported Operating Systems

- ✅ Windows
- ❌ macOS
- ❌ Linux
