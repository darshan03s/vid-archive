# VidArchive

VidArchive is a desktop utility application for downloading video and audio through an interactive user interface.

Media extraction is handled via integration with yt-dlp and ffmpeg.

[![Watch the demo](https://res.cloudinary.com/dqzusd5rw/image/upload/v1765874362/Screenshot_2025-12-16_140211_prvwl5.png)](https://res.cloudinary.com/dqzusd5rw/video/upload/v1765874331/VidArchive_0.18.1_final_demo_1_xq7sdo.mp4)

## Tech Stack

- Electron.js
- React.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- better-sqlite3
- Drizzle ORM
- Zustand

## Features

- Download video and audio media
- View media metadata
- Interactively select formats and options
- URL and download history
- Application settings

## Supported Operating Systems

- ✅ Windows
- ❌ macOS
- ❌ Linux
